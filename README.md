# 🚀 XinServer · 零代码后端服务平台

开箱即用 · 可视化建表 · 自动生成 API · 可视化管理后台 · 一键部署

**XinServer** 是专为 **前端 / App 开发者、独立开发者、创业团队** 打造的零代码后端服务平台。
无需后端基础、无需写 API 代码，只要会调用接口，就能独立完成项目开发。

🌟 **让所有开发者都具有全栈能力，把精力专注在业务实现上。**

## 🔥 平台简介

XinServer = **可视化建表** + **自动生成 API** + **内置运营后台** + **运维平台** + **一键部署**

* 无需编写 SQL / 后端代码
* 可视化字段配置，自动生成数据表、数据管理页面、API 接口
* 直接调用接口即可做 App / Web / 桌面端 / 小程序项目
* 一台服务器即可跑完整后端
* 支持 Docker 一键安装 & 私有部署

> 💡 XinServer是实打实的先进生产力，起码帮你节省掉后端开发的时间和成本，测试质量远超普通后端程序猿成果。

## 📦 功能清单

1. **数据表管理**: 可视化建表，配置字段、索引及虚拟字段，自动生成API。
2. **用户管理**: 管理系统用户与外部用户，支持新增、禁用、重置密码等操作。
3. **部门管理**: 配置组织结构，支持部门、子部门及层级管理。
4. **用户标签**: 为用户标记标签，用于筛选、分组与运营管理。
5. **用户角色**: 配置角色权限，控制不同角色的操作范围。
6. **用户权限**: 细粒度权限控制，定义角色可用的菜单与接口权限。
7. **数据字典**: 维护系统常用选项数据，用于表单选择、标签、多选项等。
8. **参数配置**: 存储系统参数、业务配置及临时缓存，支持JSON结构。
9. **资源库**: 统一管理上传的图片、文件资源，支持分组与预览。
10. **公告管理**: 发布系统公告，支持标题、内容与展示控制。
11. **消息通知**: 向用户发送站内通知，支持跳转链接与批量发送。
12. **日志管理**: 记录用户操作日志与系统行为，方便追踪排查。
13. **拓展授权**: 生成开放接口 Token，用于APP或前端未登录状态调用接口。
14. **运维管理**: 查看系统状态，一键重启服务，管理系统运行环境。
15. **系统操作**: 提供后台核心系统操作，如刷新配置、服务维护等。
16. **数据库备份**: 手动或定期备份数据库，支持下载与恢复。
17. **项目备份**: 导出项目全量数据，用于迁移或完整备份。
18. **三方服务**: 配置短信、邮箱、七牛云等第三方服务能力。
19. **插件管理**: 扩展系统功能，管理各类插件模块。
20. **模板市场**: 提供数据表、字典、标签等预设模板，一键导入使用。
21. **数据表文档**: 自动生成表结构说明，用于开发者快速查询。
22. **接口文档**: 自动生成API文档，支持调试、自定义参数与查看响应。
23. **文档中心**: 访问完整说明文档、安装指南与使用教程。
